# ميديكال زون — دليل الرفع والتعديل

## رفع التحديثات على GitHub Pages

1. انسخ محتويات مجلد `medical-zone-main` إلى مستودع GitHub.
2. من **Settings → Pages**: المصدر `main` / `/ (root)`.
3. تأكد من وجود ملف `CNAME` يحتوي: `medzon.store`
4. في إعدادات الدومين عند المسجّل: فعّل **Enforce HTTPS**.

## تعديل رقم واتساب (مهم)

افتح `js/config.js` وغيّر:

```javascript
whatsapp: {
  default: '201223962139',  // بدون + أو 0 في البداية — صيغة دولية
}
```

## إضافة المنتجات

1. أضف المنتجات في `content/products.json` (انظر `content/products.example.json`).
2. ضع صور المنتجات في `images/products/`.
3. كل منتج يحتاج `id` فريداً يُستخدم في الرابط: `product.html?id=xxx`

## الصفحات الجديدة

- `terms.html` — الشروط
- `privacy.html` — الخصوصية
- `faq.html` — أسئلة شائعة
- `support.html` — دعم فني

## ملاحظات

- لا يوجد دفع إلكتروني — الطلبات عبر واتساب فقط.
- النماذج (تواصل / صيانة) تفتح واتساب برسالة جاهزة.
- زر واتساب ثابت يظهر في كل الصفحات تلقائياً.
